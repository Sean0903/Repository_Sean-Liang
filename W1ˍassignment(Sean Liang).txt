答案：d. val array = Array(6) { 1000.0.pow( it ) }
因為「Array(6)」是代表共有六筆資料，「1000.0.pow( it )」代表從第一筆數字開始，每一筆資料會乘與1000，在第一筆數字設定為１的情況下，第二筆就會是1000，第三筆會是1000000，以此類推。
